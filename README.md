# music
web音乐播放器  
api使用了[NeteaseCloudMusicApi](https://binaryify.github.io/NeteaseCloudMusicApi), 感谢Binaryify  
个人制作, 只用于学习和交流. 如要用作其余用途, 请立即删除
## 安装和运行
请先安装和启动NeteaseCloudMusicApi, 再执行以下
```
git clone 
npm install
npm run serve
```
